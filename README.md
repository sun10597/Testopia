# Testopia
**DB 기반 성향 테스트 웹 서비스**

Spring Boot와 OAuth2를 활용해  
소셜 로그인과 동적 성향 테스트 기능을 구현한 개인 웹 프로젝트입니다.

## 📌 프로젝트 개요

- **기간**: 2025.11.05 ~ 2025.12.12  
- **유형**: 개인 사이드 프로젝트 (웹 서비스)  
- **역할**: 기획 · 설계 · 프론트엔드 · 백엔드 · DB 전 과정  
- **GitHub**: https://github.com/sun10597/Testopia  

---

## 🛠 기술 스택

- **Backend**: Spring Boot, Spring Security, JPA, OAuth2
- **Frontend**: Mustache, Tailwind CSS
- **Database**: MySQL, **AWS RDS**
- **Build**: Gradle
- **JDK**: 23

---

## ✨ 주요 기능

- Google / Kakao / Naver OAuth2 소셜 로그인
- DB 기반 테스트 · 문항 · 선택지 동적 로딩
- 한 문제씩 진행되는 성향 테스트 UI
- 문항 점수 합산 기반 결과 계산 및 저장
- 관리자 회원 / 게시글 관리

---

## 🏆 핵심 포인트

- DB 중심 설계로 테스트 확장성 확보
- OAuth2 인증 흐름 실서비스 수준 구현
- AWS RDS(MySQL) 기반 외부 DB 연동 경험
- 개인 프로젝트 전 과정 직접 수행
